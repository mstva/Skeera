## Skeera
Android App for Keeping Score of Popular Sports Games

#### Requirement:
- Kotlin
- Android Studio

#### Installing and Running:
- Clone the repo: `git clone https://github.com/mstva/Skeera.git`
- Open the project in Android Studio
- Build and Run
